使用步骤

  • 1. 选择操作类型：加密PDF或解密PDF
  • 2. 加密操作：设置打开密码、权限密码和权限选项
  • 3. 解密操作：输入PDF文件的密码
  • 4. 点击"选择文件"按钮上传PDF文件
  • 5. 点击"开始加密"或"开始解密"按钮进行处理
  • 6. 处理完成后，系统会自动下载处理后的PDF文件
