使用步骤

  • 1. 上传文件：支持多种编程语言和配置文件格式，可批量上传
  • 2. 系统自动识别源文件格式
  • 3. 选择目标格式：系统会根据源格式显示可用的目标格式
  • 4. 设置转换选项：是否格式化代码、保留注释、优化代码等
  • 5. 点击"开始转换"按钮进行处理
  • 6. 等待处理完成，系统会自动下载处理后的文件
  • 7. 查看处理结果，确保代码格式正确
