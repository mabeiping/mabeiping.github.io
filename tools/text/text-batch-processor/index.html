批量处理

批量处理文本文件

支持拖拽上传

支持格式：TXT、CSV、MD、DOC、DOCX、XLS、XLSX、RTF

处理选项

使用说明

1. 上传文件

点击上传区域或拖拽文件到页面，支持选择多个文件

2. 选择操作

从下拉菜单中选择需要的批量操作类型

3. 配置参数

根据选择的操作类型，输入相应的目标文本或参数

4. 开始处理

点击"开始处理"按钮，系统将自动处理所有文件

5. 下载结果

处理完成后，点击下载按钮获取处理后的文件

支持的操作

替换文本：将文件中所有匹配的文本替换为新文本
添加前缀/后缀：在每行文本开头或结尾添加指定内容
删除行：删除包含指定文本的行
转大写/小写：将所有文本转换为大写或小写
去除空格：删除每行首尾的空白字符

注意事项

  • 处理后的文件会自动添加"processed_"前缀
  • 大文件处理可能需要较长时间，请耐心等待
  • 建议先备份原始文件，避免数据丢失
  • 所有处理均在本地完成，不会上传到服务器
